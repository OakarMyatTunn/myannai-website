GETTING STARTED

စတင်အသုံးပြုနည်း

MyannAI tools run entirely in your browser. No account, no download, no installation needed. The only thing you need is a free Gemini API key from Google.

MyannAI ကိရိယာများသည် သင့် browser တွင်သာ လည်ပတ်သည်။ အကောင့်ဖွင့်ရန်၊ download လုပ်ရန် မလိုပါ။ Google မှ အခမဲ့ Gemini API key တစ်ခုသာ လိုအပ်သည်။

ℹ️
Your files and text are sent directly from your browser to Google's Gemini API. MyannAI never stores your data.
သင့်ဖိုင်များနှင့် စာသားများကို browser မှ တိုက်ရိုက် Google Gemini API သို့ ပေးပို့သည်။ MyannAI တွင် သင့်ဒေတာ သိမ်းဆည်းမည်မဟုတ်ပါ။
🔑 Required

GEMINI API KEY SETUP

Gemini API Key ရယူပုံ
✅
Gemini API is completely FREE for personal use. No credit card required.
Gemini API သည် ကိုယ်ရေးကိုယ်တာ အသုံးပြုရန် လုံးဝ အခမဲ့ဖြစ်သည်။ ခရက်ဒစ်ကဒ် မလိုအပ်ပါ။
01
Go to aistudio.google.com and sign in with your Google account.
aistudio.google.com သို့ သွားပြီး Google အကောင့်ဖြင့် ဝင်ရောက်ပါ
02
Click "Create API Key" then select "Create API key in new project".
"Create API Key" ကိုနှိပ်ပြီး "Create API key in new project" ကို ရွေးပါ
03
Copy the key that starts with AIzaSy... — it's a long string of letters and numbers.
AIzaSy... ဖြင့် စတင်သော key ကို ကူးယူပါ
04
Paste it into the API Key field on any MyannAI tool page and click Save.
MyannAI tool page ရှိ API Key field တွင် ကူးထည့်ပြီး Save နှိပ်ပါ
⚠️
Never share your API key publicly. It's like a password — keep it private.
API key ကို အများသိအောင် မမျှဝေပါနှင့်။ Password ကဲ့သို့ လျှို့ဝှက်ထားပါ။
🎙️ Tool 1

TRANSCRIPT AI

အသံမှ စာသားသို့ ပြောင်းပေးသော ကိရိယာ

Converts any audio or video file into accurate text. Can also translate the transcript into a different language.
မည်သည့် အသံ သို့မဟုတ် ဗီဒီယိုဖိုင်ကိုမဆို တိကျသော စာသားအဖြစ် ပြောင်းလဲပေးသည်။ ဘာသာပြန်ဆိုလည်း ပြုလုပ်နိုင်သည်။

1
Enter your Gemini API Key and click Load
Gemini API Key ကို ထည့်ပြီး Load နှိပ်ပါ
Paste your key from aistudio.google.com. Click Load to verify it works. You'll see a green confirmation message.
aistudio.google.com မှ key ကို ကူးထည့်ပါ။ Load နှိပ်ပြီး အလုပ်လုပ်မလုပ် စစ်ဆေးပါ။
2
Select Audio Language and Output Language
အသံဘာသာစကားနှင့် ထွက်ဘာသာစကား ရွေးပါ
Audio Language = the language spoken in the file. Output Language = the language you want the transcript in. Set to "Same as Audio" if no translation needed.
Audio Language = ဖိုင်ထဲမှ ပြောဆိုသော ဘာသာစကား။ Output Language = transcript ထွက်လိုသော ဘာသာစကား။
3
Upload your audio or video file
အသံ သို့မဟုတ် ဗီဒီယိုဖိုင် တင်ပါ
Click the upload area or drag and drop your file. Supported formats: MP4, MP3, WAV, M4A, MOV, AVI. Maximum file size: 2GB.
Upload နေရာကို နှိပ်ပါ သို့မဟုတ် ဖိုင်ကို ဆွဲချပါ။ MP4, MP3, WAV, M4A, MOV ပုံစံများ ထောက်ပံ့သည်။
4
Click "Start Transcription" and wait
"Start Transcription" နှိပ်ပြီး စောင့်ပါ
Processing time depends on file size. A 10-minute audio takes roughly 30–60 seconds. Do not close the browser tab while processing.
ဖိုင်အရွယ်အစားပေါ်မူတည်၍ အချိန်ကြာနိုင်သည်။ browser tab ကို မပိတ်ပါနှင့်။
5
Copy or download your transcript
Transcript ကို ကူးယူ သို့မဟုတ် download လုပ်ပါ
Switch between Transcript and SRT tabs. Use Copy Text to copy to clipboard, or Download SRT for subtitle files, Download TXT for plain text.
Transcript နှင့် SRT tab ကြားတွင် ပြောင်းနိုင်သည်။ Copy Text, Download SRT, Download TXT ခလုတ်များ အသုံးပြုနိုင်သည်။
🎬 Tool 2

RECAPPER AI

ဗီဒီယို အကျဉ်းချုပ် ကိရိယာ

Watches your video and generates a compelling story recap — perfect for social media sharing.
ဗီဒီယိုကြည့်ပြီး ဆွဲဆောင်မှုရှိသော ဇာတ်လမ်းအကျဉ်းချုပ် ထုတ်ပေးသည်။

1
Enter your API Key and click Save
API Key ထည့်ပြီး Save နှိပ်ပါ
Same key used for Transcript AI. Once saved you don't need to re-enter it on the same visit.
2
Choose Output Language
ထွက်ဘာသာစကား ရွေးပါ
Myanmar only, English only, or Both. "Both" generates a full recap in English followed by Myanmar — useful for bilingual social media posts.
မြန်မာသာ၊ အင်္ဂလိပ်သာ သို့မဟုတ် နှစ်မျိုးလုံး ရွေးနိုင်သည်။
3
Upload your video and click Generate
ဗီဒီယို တင်ပြီး Generate နှိပ်ပါ
Best results with movies, documentaries, lectures, or YouTube videos saved as MP4. The AI reads the visual content, dialogue, and context.
ရုပ်ရှင်၊ မှတ်တမ်းဗီဒီယို၊ သင်တန်းများ၊ YouTube ဗီဒီယိုများတွင် အကောင်းဆုံး ရလဒ်ရသည်။
4
Copy and share the recap
Recap ကို ကူးယူ၍ မျှဝေပါ
Copy the recap for TikTok, Facebook, or YouTube descriptions. Or download as TXT for editing in any word processor.
TikTok, Facebook, YouTube description များတွင် တိုက်ရိုက် ကူးထည့်နိုင်သည်။
💡
For best recap quality: use videos with clear dialogue, not music videos. Movies and lectures work best.
အကောင်းဆုံး ရလဒ်ရရန်: ဒိုင်ယာလော့ရှင်းသော ဗီဒီယိုများ အသုံးပြုပါ။ ဂီတဗီဒီယိုများ သင့်တော်သည်မဟုတ်ပါ။
🔊 Tool 3

TTS AI

စာသားမှ အသံသို့ ပြောင်းပေးသော ကိရိယာ

Converts any text into natural-sounding MP3 audio. Great for voiceovers, accessibility, and content creation.
မည်သည့်စာသားကိုမဆို သဘာဝကျသော MP3 အသံဖိုင်အဖြစ် ပြောင်းလဲပေးသည်။

1
Enter API Key and click Save
Same Gemini API key from aistudio.google.com.
2
Choose Language and Voice Style
ဘာသာစကားနှင့် အသံပုံစံ ရွေးပါ
Natural — everyday speech.
Storytelling — expressive, emotional, great for recaps.
News Report — formal, clear, professional.
Commercial — energetic, persuasive.
Natural = သဘာဝပြောဆိုမှု။ Storytelling = ဇာတ်လမ်းပြောရန်။ News = ရင်ရင်ဝင်ဝင်တင်ပြမှု။
3
Type or paste your text
စာသား ရိုက်ထည့် သို့မဟုတ် ကူးထည့်ပါ
Supports English and Myanmar text. You can mix both languages in the same text box. The character counter shows how long your text is.
အင်္ဂလိပ်နှင့် မြန်မာဘာသာ နှစ်မျိုးလုံး ထောက်ပံ့သည်။ နှစ်မျိုးကို ရောစပ်၍ရသည်။
4
Click Transform and download MP3
Transform နှိပ်ပြီး MP3 download လုပ်ပါ
Audio plays directly in the browser. Click Download MP3 to save the file to your device. Use in videos, podcasts, or any content.
Browser တွင် တိုက်ရိုက် နားထောင်နိုင်သည်။ Download MP3 နှိပ်၍ ဖိုင်သိမ်းနိုင်သည်။

❓ FAQ

FREQUENTLY ASKED QUESTIONS

မကြာခဏ မေးလေ့ရှိသောမေးခွန်းများ
Is MyannAI completely free?
MyannAI သည် လုံးဝ အခမဲ့လား?
▼
Yes. MyannAI itself is free. You bring your own Gemini API key from Google, which has a generous free tier — enough for hundreds of requests per day for personal use.

ဟုတ်ပါသည်။ MyannAI ကိုယ်တိုင် အခမဲ့ဖြစ်သည်။ Google မှ Gemini API key ကိုသာ ကိုယ်တိုင် ယူဆောင်လာရသည်။ ၎င်းသည်လည်း ကိုယ်ရေးကိုယ်တာ အသုံးပြုမှုအတွက် တစ်နေ့လျှင် တောင်းဆိုချက် ရာနှင့်ချီ၍ အသုံးပြုနိုင်သော အခမဲ့ tier ရှိသည်။
Are my files stored on your servers?
ဖိုင်များကို ဆာဗာတွင် သိမ်းဆည်းမည်လား?
▼
No. MyannAI has no servers. Your files are sent directly from your browser to Google's Gemini API and are not stored anywhere by us. Check Google's privacy policy for how they handle uploaded data.

မဟုတ်ပါ။ MyannAI တွင် ဆာဗာမရှိပါ။ ဖိုင်များကို browser မှ တိုက်ရိုက် Google Gemini API သို့ ပေးပို့သည်။ ကျွန်ုပ်တို့ ဘာကိုမျှ သိမ်းမထားပါ။
Why did I get an error "quota exceeded"?
"quota exceeded" အမှားပြပါသလား?
▼
Your Gemini API free tier has daily limits. If you hit the limit, wait until the next day — it resets automatically at midnight. You can also create a new API key in a new Google project to get fresh quota.

Gemini API free tier တွင် နေ့စဉ် ကန့်သတ်ချက် ရှိသည်။ ကန့်သတ်ချက်ရောက်ပါက နောက်တစ်ရက် ညသန်းခေါင်တွင် အလိုအလျောက် ပြန်လည်ချိန်ညှိသည်။
What languages are supported for transcription?
မည်သည့်ဘာသာစကားများ ထောက်ပံ့သနည်း?
▼
Gemini supports over 40 languages for transcription including English, Myanmar (Burmese), Thai, Chinese, Japanese, Korean, and most major world languages. Use "Auto Detect" if you're unsure.

Gemini သည် အင်္ဂလိပ်၊ မြန်မာ၊ ထိုင်း၊ တရုတ်၊ ဂျပန်၊ ကိုရီးယားစသော ဘာသာစကား ၄၀ ကျော် ထောက်ပံ့သည်။ မသေချာပါက "Auto Detect" ရွေးပါ။
How long does processing take?
လုပ်ဆောင်ချိန် မည်မျှ ကြာသနည်း?
▼
Depends on file size and your internet connection. A typical 10-minute video takes 30–90 seconds. A 2-hour movie may take 3–5 minutes. Keep the browser tab open until complete.

ဖိုင်အရွယ်အစားနှင့် internet အမြန်နှုန်းပေါ်မူတည်သည်။ မိနစ် ၁၀ ဗီဒီယိုသည် ၃၀–၉၀ စက္ကန့်ကြာသည်။ နှစ်နာရီရုပ်ရှင်သည် ၃–၅ မိနစ်ကြာနိုင်သည်။
Can I use these tools on my phone?
ဖုန်းတွင် အသုံးပြုနိုင်သလား?
▼
Yes. All tools work on mobile browsers (Chrome, Safari, Firefox). File upload from phone storage is supported. Note that large files may take longer on mobile connections.

ဟုတ်ပါသည်။ ကိရိယာများအားလုံး မိုဘိုင်း browser (Chrome, Safari, Firefox) တွင် အလုပ်လုပ်သည်။ ဖုန်းမှ ဖိုင်တင်ခြင်းကို ထောက်ပံ့သည်။
